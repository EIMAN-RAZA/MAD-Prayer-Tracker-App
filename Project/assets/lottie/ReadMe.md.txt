# Lottie Animations for Prayer Tracker App

This folder should contain Lottie animation files (.json).

For this app to work with animations, please download the following animations:

1. **fire.json** - Streak animation
   Download from: https://lottiefiles.com/search?q=fire&category=animations

2. **prayer.json** - Prayer animation  
   Download from: https://lottiefiles.com/search?q=prayer&category=animations

3. **quran.json** - Quran book animation
   Download from: https://lottiefiles.com/search?q=book&category=animations

OR use any of these alternatives:

- Use Flare animations instead
- Replace with GIF images
- Use built-in Flutter animations (already implemented in code)

The app will work without these files - it will use built-in Flutter animations as fallback.
